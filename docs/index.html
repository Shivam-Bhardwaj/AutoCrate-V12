üöÄ Quick Start Guide

Prerequisites

  • Python 3.8+ - Modern Python interpreter
  • Windows OS - Primary development and testing platform
  • Siemens NX - For CAD model generation (optional for calculations)

Installation Steps

1. Repository Setup

# Clone the repository
git clone https://github.com/your-username/autocrate.git
cd autocrate

# Verify Python version
python --version  # Should be 3.8 or higher

2. Virtual Environment

# Create virtual environment
python -m venv autocrate-env

# Activate environment
# Windows:
autocrate-env\Scripts\activate
# macOS/Linux:
source autocrate-env/bin/activate

3. Dependencies Installation

# Install required packages
pip install -r requirements.txt

# Verify installation
pip list | grep -E "(tkinter|pytest|coverage)"

4. Launch Application

# GUI Mode (Recommended)
python legacy/nx_expressions_generator.py

# Or through module
python -m autocrate

# Command Line Mode
python legacy/plywood_layout_generator.py --help
üí° Pro Tip: Use the GUI mode for interactive design and the command line for batch processing or automation workflows.

üìã Design Workflow

AutoCrate follows a sophisticated, multi-stage workflow that ensures accurate, manufacturable crate designs:

Input Specification Collection

Captures comprehensive product and design requirements:

  • Product dimensions (length √ó width √ó height)
  • Product weight and weight distribution
  • Clearance requirements (all sides, top, bottom)
  • Material specifications (plywood thickness, cleat sizes)
  • Special requirements (access panels, lifting points)

Structural Analysis & Skid Sizing

Determines foundation requirements based on load analysis:

  • Calculates required lumber size (3√ó4, 4√ó4, 4√ó6, 6√ó6, 8√ó8)
  • Determines maximum skid spacing for load distribution
  • Validates against shipping and handling requirements
  • Optimizes for standard lumber availability

Panel Dimension Calculation

Calculates initial panel sizes with assembly considerations:

  • Front/Back panels span full crate width
  • Left/Right end panels fit between front/back panels
  • Top panel covers entire crate footprint
  • Accounts for material thickness and assembly clearances

Iterative Dimension Stabilization

Critical process ensuring dimensional accuracy:

  • Calculates cleat material requirements for each panel
  • Adjusts overall crate dimensions to accommodate cleats
  • Recalculates affected panel dimensions
  • Repeats until all dimensions stabilize
  • Ensures final clearances meet specifications

Material Optimization

Advanced algorithms minimize waste and cost:

  • Calculates optimal plywood sheet layouts
  • Minimizes number of sheets required
  • Identifies splice positions for structural planning
  • Generates cut lists and material schedules

Structural Reinforcement Design

Places cleats according to engineering standards:

  • Mandatory cleats at all plywood splice locations
  • Additional cleats to maintain 24" maximum spacing
  • Edge cleats for panel perimeter reinforcement
  • Optimized cleat lengths to minimize waste

CAD Data Generation

Produces complete parametric CAD model data:

  • Generates Siemens NX expressions file (.exp)
  • Includes all dimensions, positions, and part counts
  • Sets suppression flags for unused components
  • Enables automatic 3D model and drawing generation
‚úÖ Quality Assurance: Each step includes validation checks to ensure the resulting design is manufacturable, structurally sound, and meets all input requirements.
